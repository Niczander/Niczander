# Handover Report: U Home Supermarkets Platform

## 1. Project Overview

- **Project Name**: U Home Supermarkets E-commerce Platform
- **Purpose**: An online store with a comprehensive admin panel for managing products, orders, and site content.
- **Technology Stack**: PHP, MySQL, Bootstrap.

## 2. System Architecture

- **`admin/`**: Contains the backend administration panel.
- **`assets/`**: Holds CSS, JavaScript, and images.
- **`database/`**: Includes the `schema.sql` for database setup.
- **`includes/`**: Contains shared components like the database connection, header, and footer.
- **Root Files**: Frontend pages like `index.php`, `cart.php`, etc.
